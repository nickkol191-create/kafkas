ΚΑΥΚΑΣ — Product photos
========================

Drop your product photos here (white background, square works best — e.g. 800x800).
Then open  assets/js/data.js  and add one line per photo inside PRODUCT_PHOTOS, e.g.:

    const PRODUCT_PHOTOS = {
      av1: 'av1.jpg',
      sh1: 'sh1.webp',
      ...
    };

(Any format works: .jpg / .png / .webp. The photo replaces the icon automatically.)
Tip: just name each file after its product id below (av1.jpg, sh1.jpg ...) and tell me —
I'll fill in the whole PRODUCT_PHOTOS map for you in one go.
